# 🚀 DevOps Monitoring Project

## 📌 Overview
This project is a simple DevOps Monitoring System built using Python Flask and MySQL, containerized using Docker and Docker Compose.

---

## 🛠️ Tech Stack
- Python (Flask)
- MySQL
- Docker
- Docker Compose
- Linux (WSL)

---

## ⚙️ Features
- Containerized Flask app
- MySQL integration
- Dynamic logs
- Multi-container setup

---

## 🚀 How to Run

git clone https://github.com/vijaymohalkar00-cmyk/monitoring-devops-project
cd monitoring-devops-project
docker-compose up --build

Open: http://localhost:5000
